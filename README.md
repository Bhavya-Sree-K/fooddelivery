## 📌 Project Overview

A fully responsive, interactive **Food Delivery Web Application** showcasing authentic Indian cuisine. The app features a beautifully designed landing page, dynamic menu browsing, restaurant listings, and a seamless cart experience — all built with pure HTML, CSS, and Vanilla JavaScript.

---

## ✨ Features

- 🏠 **Hero Landing Page** — Stunning food imagery with a bold call-to-action
- 🍛 **Dynamic Menu System** — Browse dishes by category with real-time filtering
- 🏪 **Restaurant Listings** — View available restaurants with ratings and cuisine type
- 🛒 **Shopping Cart** — Add/remove items with live price calculation
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet, and desktop
- 🎨 **Premium Animations** — Smooth hover effects, transitions, and micro-animations
- 🌙 **Modern UI** — Clean card-based layout with warm, food-inspired color palette

## 🚀 Getting Started

### View Live
Simply open the live link: [fooddeliveryappforpeople.netlify.app](https://fooddeliveryappforpeople.netlify.app)

### Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/Bhavya-Sree-K/fooddelivery.git
   ```
2. Open `index.html` in your browser — no server or build step required!
